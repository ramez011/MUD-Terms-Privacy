1. Introduction
This Privacy Policy explains what data the MUD Discord bot (“the Bot”) collects, how it is used, and how it is stored.

2. Data the Bot Collects
The Bot does not collect or store any personal user data such as:

Usernames

User IDs

Messages

IP addresses

Emails

Profile information

The Bot only stores server‑specific data that server administrators manually enter, including:

Location names, postal codes, and optional image URLs

Crafting item names, materials, and optional image URLs

This data is stored in local JSON files on the bot owner’s system.

3. How Data Is Used
The stored data is used solely to provide the Bot’s features, such as:

Displaying saved locations

Displaying crafting recipes

Autocomplete suggestions

Dropdown menus

The Bot does not analyze, sell, or share any data with third parties.

4. Data Storage
All data is stored locally on the bot owner’s machine in files named:
serverData_<guildId>.json  
No data is transmitted to external servers or third‑party services.

5. Data Removal
Server owners may request deletion of their server’s stored data at any time by contacting the bot owner at zjordan011@gmail.com.
Data will be deleted within a reasonable timeframe.

6. Children’s Privacy
The Bot does not knowingly collect any personal information from children under 13. If you believe data has been collected improperly, contact us immediately.

7. Changes to This Policy
This Privacy Policy may be updated at any time. Continued use of the Bot after changes means you accept the updated policy.

8. Contact
For privacy concerns or data deletion requests, contact: zjordan011@gmail.com
